## 제어문
(키워드 : 제어문, 반복문, 함수, 클래스)

1. if 조건문
```
if == '초록' :
    건넌다()
```
- if 다음 비교 조건이 T일때는 조건이 맞는다, 참이다, 존재한다.
- if 다음 비교 조건이 F일때는 조건이 맞지않는다, 거짓이다, 존재하지 않는다.
- 조건이 부합하면 : 다음으로 기능을 하는 비즈니스 로직 코드로 넘어간다.
- 조건을 진행해서 이에 부합할시 넘어가는 코드의 들여쓰기는 4칸으로 고정된다(indent)
, 2칸으로 쓰고자 하면 전체 포맷을 2칸으로 바꿔야 한다.

4. 조건문(if, elif,else)
- if의 조건이 맞지 않을때 다음 조건 elif, if-elif 아무런 조건도 맞지 않을때 else
- 비즈니스 로직의 개발에 조건문을 쓸시 문서로 명문화 시켜놓고 개발을 하는것이 개발에 용이
    * 65세 이상은 무료입장, 13-64까지는 3만원, 3-12까지는 만원 - 이렇게 명문화 시켜서 만들어야 나중에 코드짤때 유리
- 클린코드를 적용하는 회사 같은경우 Early Exit(불필요하게 조건문을 중첩화 하거나, 길게 만들지 않고 조건이 맞지 않으면 return,break로 바로 탈출하게 만드는 로직)
룰에 따라 모든 조건을 검토하고 실행하는 else문을 잘 쓰지않는 경우도 있다.
- 조건문의 실행을 위해 indent를 확인하는 습관을 들인다. (tap키로 indent이동, shift+tap으로 뒤로 이동)
- if의 조건을 비교하는 방법 --> 비교연산자 '==' 를 사용(다른 비교연산자 : >,<,>=,<=,!=, ) 
- '=' 이 연산자는 변수에 값을 할당하는 할당연산자 / '==' 값을 비교하는 비교연산자

1. for문 - 반복적 계산을 진행, 순차적으로 진행
- range() - 함수 사용하여 특정 숫자만 순회하여 반복
- enumerate() - for를 돌리는 list 인덱싱에 순번을 매겨주는 역할, 
```
enumerate() 시나리오
my_list = ['선영', '지수', '영희', '철수']

for i, l in enumerate(my_list,start=1):
    print(i,l)
    
- enumerate()를 사용하여 깔끔하게 리스트의 인덱스 값과, 그 인덱스의 순번 까지 출력
- enumerate()는 해당리스트와, 그 리스트의 순번을 매개로 받아 튜플로 반환
- for의 i,l 변수로 튜플안의 각각의 변수를 받고, 이를 출력 가능

num = 1
for i in my_list:
    print(f"{num}: {i}")
    num = num + 1
```

## 자료형(Data Type)
변수에 저장된 값의 종류의 형식을 나타내는것, 파이썬은 저장값의 형식을 자동으로 인식한다.

- int - 데이터 타입중 정수형 숫자를 가진 값이며, 아라빅 숫자(1,-3,0)로 표기된다
- float - 소수형 숫자를 가진 값, 정수와 이후 . 소숫점이 존재하는 값
- str - 문자, 단어 등으로 구성된 문자들의 집합체의 
- bool - Ture, False 2가지 값을 가지며 진실 혹은 거짓을 판별하는 값으로 사용
- list - []안에 다수의 데이터를 묶어서 사용하는 용도로, 리스트 안에 들어가는 값은 각각의 index로 구분되어 개별 출력과 수정이 가능하다.
- tuple - ()안에 다수의 데이터를 묶어서 사하는 용도, 인덱싱이 가능하다 해당값의 수정은 불가하다.
(1)은 정수값으로 한가지의 인덱스를 가진 튜플로 만들시 (1,)로 만들어야 된다.
- dict - {키:값}안에 키와 값을 매치시켜 순차적인 인덱스를 요구 하지 않고 각각의 키를 통해 바로 자료를 출력가능
- set - {}안에 중복된 값이 없는 유일값의 집합체로 순서가 없는 데이터 구조로 인덱싱화 할수 없다.

## 조건문과 반복문 (if, for, while)
링크(https://rebro.kr/130)

조건문은 특정 조건을 만족할때만 코드를 수행하는 문법
- if - 입력한 조건을 만족하는 경우에만 다음 조건의 코드를 수행하는 명령어
- elif - if의 조건이외에 여러 조건을 더 추가 하고 싶을떼 사용
- else - 작성한 조건들이 모두 만족하지 않을때 수행하는 코드

반복문은 반복되는 작업을 일정한 코드로 간단하게 수행하기 위한 코드
- for - 정해진 범위를 순회하며 반복을 실행, 반복횟수가 정해져있을때 사용하면 유용
- while - 조건이 '참'인동안 계속 반복을 실행, 반복획수 보다 조건에 따라 반복을 실행할시 유용

## 함수 정의와 호출

함수(Funtion) - 특정 작업을 수행하기 위해 만들어 놓은 코드의 집합을 의미하며, 특정 기능의 코드를 반복작성 할 필요 없이 함수화 하여 기능을 수행하고, 결과 값을 반환할수 있어 코드 가독성에 유용하다

파이썬에서는 def 키워드로 함수를 정의하며, def 함수이름()로 함수를 생성하며 ()안에는 함수가 변수값을 받을수 있는 매개 변수를 입력한다(매개변수가 없을시 공란으로 정의한다.)

함수를 호출하여 사용하기 위해서는 함수의 이름과 ()를 붙여 호툴하면 된다.

## 기본 입출력 (input, print)

- input은 파이썬 내장함수중 하나로 사용자의 입력 정보를 받아 처리하는 역할을 한다. input()로 호출하며 입력값을 변수에 저장이 가능하다.
- print는 파이썬 내장함수중 하나로 변수의 값뿐만 아니라, 결과값을 화면에 출력해주는 역활을 한다.

## 클래스에 대하여
객체를 사용자가 정의,생성하여 사용하기 위한 문법, 사용자 정의 객체와 그 객체를 동작하는 함수를 묶어 하나의 집합채로 만들수 있다.

Class 이름: 으로 클래스를 만들수 있으며 정의된 클래스를 객체로서 사용하기 위해서는 일정한 이름을 가진 변수에 담아 사용한다.
